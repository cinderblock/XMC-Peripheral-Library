Detailed Description

Defines a data structure for initializing the data of the supply voltage monitoring block. Supply voltage monitoring block consists of 2 detectors namely External voltage detector (VDEL) and External brownout detector (BDE) in the EVR that are used to monitor the VDDP. VDEL detector compares the supply voltage against a pre-warning threshold voltage. Use type XMC_SCU_SUPPLYMONITOR_t for accessing these structure parameters.

uint32_t ext_supply_monitor_speed

Speed of the voltage monitor(VDEL Timing Setting).
Range:
00B sets monitor speed typ 1µs - slowest response time, 01B sets monitor speed typ 500n, 10B sets monitor speed typ 250n, 11B sets monitor speed with no delay - fastest response time.

uint32_t ext_supply_threshold

External supply range (VDEL Range Select).
Range: 00B sets threshold value to 2.25V, 01B sets threshold value to 3.0V and 10B sets threshold value to 4.4V
