XMC_ACMP_CONFIG_t Struct Reference
XMC Peripheral Library » ACMP

#include <xmc_acmp.h>

Detailed Description

Structure for initializing the ACMP module. It configures the ANACMP register of the respective input.

Field Documentation

uint32_t filter_disable

Comparator filter option for removing glitches. By default this option is selected in ANACMP register. Setting this option disables the filter

uint32_t hysteresis

Hysteresis voltage to reduce noise sensitivity. Select the voltage levels from the values defined in XMC_ACMP_HYSTERESIS_t.

uint32_t output_invert

Option to invert the comparator output.
